Gas Station Recommendation App

AI-powered gas station finder with real-time pricing and smart recommendations

Find Nearby Stations

Locate gas stations near you or any address

AI Recommendations

Get smart suggestions based on cost and convenience

Price Comparison

Compare prices across different fuel grades

Key Features

Current Location

Use your GPS location or enter any address to find nearby gas stations.

Multiple Fuel Grades

Compare prices for Regular (87), Mid-Grade (89), and Premium (91) fuel.

Cost Analysis

Calculate total costs including fuel and travel expenses.

Deploy Your Own Instance

Quick Deploy
Deploy to Heroku:
git clone https://github.com/yourusername/gas-station-recommendation.git
cd gas-station-recommendation
heroku create your-app-name
heroku config:set GOOGLE_MAPS_API_KEY=your_key
heroku config:set CLAUDE_API_KEY=your_key
git push heroku main
Docker Deploy
Deploy with Docker:
git clone https://github.com/yourusername/gas-station-recommendation.git
cd gas-station-recommendation
docker build -t gas-app .
docker run -p 8080:8080 gas-app
